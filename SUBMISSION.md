## 📖 Game Introduction

### 🎯 Overview
**SlimeGobble** is a 2D action puzzle game inspired by the classic Pac-Man. Players control an adorable slime named "Mumu" to collect coins in a maze while avoiding intelligent chasers.

### 🏆 Game Objectives
- 🪙 Collect **500 points** per level to advance
- 🎮 Complete all **3 levels** to win
- ❤️ Avoid enemies (3 lives total)

### Controls
| Key | Function |
|-----|----------|
| W/A/S/D | Move Up/Left/Down/Right |
| P | Pause/Resume |
| ESC | Return to Main Menu |
